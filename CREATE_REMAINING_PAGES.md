# Remaining Pages to Create

This document lists all the pages that still need to be created for the Tap And Buy eCommerce platform.

## Customer Pages
1. Categories.tsx - Browse products by category
2. ProductDetail.tsx - View product details and add to cart
3. Cart.tsx - View cart items and proceed to checkout
4. Checkout.tsx - Select address and review order
5. Payment.tsx - UPI QR code payment with reference number
6. Account.tsx - User profile and settings
7. Orders.tsx - View order history
8. OrderDetail.tsx - View order details and tracking
9. Addresses.tsx - Manage delivery addresses
10. Support.tsx - Customer support chatbot

## Admin Pages
1. Dashboard.tsx - Admin overview
2. Products.tsx - Manage products
3. Categories.tsx - Manage categories
4. Orders.tsx - Manage all orders
5. Returns.tsx - Manage return requests
6. Promotions.tsx - Manage promotional images

## Implementation Priority
1. Cart, Checkout, Payment (critical for purchases)
2. ProductDetail, Categories (browsing)
3. Account, Orders, OrderDetail (user management)
4. Admin pages (management)
5. Support (customer service)

## Notes
- All pages need proper error handling
- All pages need loading states
- All pages need to be responsive
- Admin pages need role checking
